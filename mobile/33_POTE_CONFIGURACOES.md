# Tela de Configurações do Pote (Assinaturas)

## Visão Geral

Configurações do sistema de assinaturas/pote.

## Layout

- Lista de configurações
- Botão "Nova Configuração"
- Campos: Nome, Valor do pote, Regras de distribuição

## Rotas API

- `GET /pote/configuracoes?barbearia_id=X`
- `POST /pote/configuracoes`
- `PUT /pote/configuracoes/{id}`
- `DELETE /pote/configuracoes/{id}`

## Stores

- `usePoteStore`
