# Tela de Detalhes do Atendimento

## Visão Geral

Detalhes completos do atendimento com opção de edição/exclusão.

## Layout

- Informações do atendimento
- Lista de serviços realizados
- Lista de produtos vendidos
- Total e forma de pagamento
- Botões: Editar, Excluir, Imprimir

## Rotas API

- `GET /atendimentos/{id}`
- `PUT /atendimentos/{id}`
- `DELETE /atendimentos/{id}`
