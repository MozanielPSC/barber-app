# Tela de Canais de Marketing

## Visão Geral

Gerenciamento de gastos com canais de marketing.

## Layout

- Lista de canais com gastos
- Botão "Adicionar Canal"
- Campos: Nome do canal, Gasto mensal

## Rotas API

- `GET /financeiro/canais?barbearia_id=X&mes=YYYY-MM`
- `POST /financeiro/canais`
- `PUT /financeiro/canais/{id}`
- `DELETE /financeiro/canais/{id}`
