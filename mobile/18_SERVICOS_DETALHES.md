# Tela de Detalhes do Serviço

## Visão Geral

Detalhes do serviço, estatísticas de uso e edição.

## Layout

- Informações do serviço
- Estatísticas (quantas vezes foi usado, receita total)
- Botões: Editar, Excluir

## Rotas API

- `GET /servicos/{id}`
- `PUT /servicos/{id}`
- `DELETE /servicos/{id}`
