# Tela de Distribuições do Pote

## Visão Geral

Histórico e gerenciamento de distribuições do pote.

## Layout

- Lista de distribuições
- Filtros por período
- Cada item: Data, Valor, Destinatário, Status

## Rotas API

- `GET /pote/distribuicoes?barbearia_id=X&periodo=YYYY-MM`
- `POST /pote/distribuicoes`
