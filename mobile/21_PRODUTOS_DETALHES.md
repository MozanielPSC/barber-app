# Tela de Detalhes do Produto

## Visão Geral

Detalhes do produto, estoque, movimentações e estatísticas.

## Layout

- Informações do produto
- Estoque atual e histórico
- Estatísticas de vendas
- Botões: Editar, Excluir, Ajustar Estoque

## Rotas API

- `GET /produtos/{id}`
- `PUT /produtos/{id}`
- `DELETE /produtos/{id}`
