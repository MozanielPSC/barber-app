# Tela de Planos de Assinatura

## Visão Geral

Gerenciamento de planos de assinatura.

## Layout

- Lista de planos
- Botão "Novo Plano"
- Campos: Nome, Valor, Duração, Benefícios

## Rotas API

- `GET /pote/planos?barbearia_id=X`
- `POST /pote/planos`
- `PUT /pote/planos/{id}`
- `DELETE /pote/planos/{id}`
