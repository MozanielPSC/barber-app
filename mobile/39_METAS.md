# Tela de Configuração de Metas

## Visão Geral

Configuração de metas diárias, semanais e mensais (só proprietários).

## Layout

- Seções: Metas Diárias, Semanais, Mensais
- Para cada: Serviços, Produtos, Clientes
- Campos: Valor da meta

## Rotas API

- `GET /configuracoes/metas?barbearia_id=X`
- `PUT /configuracoes/metas`

## Stores

- `useSettingsStore`
