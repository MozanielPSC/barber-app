# Tela de Assinaturas Ativas

## Visão Geral

Lista de assinaturas ativas de clientes.

## Layout

- Lista de assinaturas
- Filtros por status, plano
- Botão "Nova Assinatura"
- Cada item: Cliente, Plano, Status, Próximo pagamento

## Rotas API

- `GET /pote/assinaturas?barbearia_id=X`
- `POST /pote/assinaturas`
- `PUT /pote/assinaturas/{id}`
- `DELETE /pote/assinaturas/{id}`
