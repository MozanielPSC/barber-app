# Tela de Gerenciamento de Barbearias

## Visão Geral

Lista e gerenciamento de barbearias (só proprietários com múltiplas barbearias).

## Layout

- Lista de barbearias
- Botão "Nova Barbearia"
- Cada item: Nome, Endereço, Status
- Botões: Editar, Excluir, Selecionar

## Rotas API

- `GET /barbearias`
- `POST /barbearias`
- `PUT /barbearias/{id}`
- `DELETE /barbearias/{id}`

## Stores

- `useBarbeariasStore`
