# Tela de Despesas Variáveis

## Visão Geral

Similar a despesas fixas, mas para despesas variáveis.

## Rotas API

- `GET /financeiro/despesas-variaveis?barbearia_id=X&mes=YYYY-MM`
- `POST /financeiro/despesas-variaveis`
- `PUT /financeiro/despesas-variaveis/{id}`
- `DELETE /financeiro/despesas-variaveis/{id}`
